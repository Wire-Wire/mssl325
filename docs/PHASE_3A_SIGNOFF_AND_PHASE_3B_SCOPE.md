## User decision 1 — Phase 3A sign-off

Phase 3A is signed off only as a descriptive, pass-aware, measurement-model-bounded Dn/EB comparison across the extant THD-only six-pass interpretable bank, under frozen encounter-averaged Sun–Earth-line s-mapping with Shue98 magnetopause, Merka05 bow shock, and propagated upstream OMNI inputs. It supports only the descriptive statement that the clean core spans Dn 0.94–2.31 and EB 0.80–1.96, while any apparent low-Dn extension below 0.5 is carried solely by cautious passes P1/P3. It does not support thresholds, labels, detector semantics, class assignment, or generalization beyond the current compressed-sheath THD bank.

## User decision 2 — Phase 3B authorization

Phase 3B is authorized only as a P1/P3 retention audit on the existing six-pass bank under the frozen measurement model. Its sole question is whether P1 and P3 may remain in the bank as cautious low-Dn comparators after explicit review of mapping sensitivity, propagated-upstream limitations, foreshock/jet alternatives, mixing risk, and boundary-motion risk. No bank expansion, no new probes or seasons, no measurement-model changes, no detector semantics, and no changes to frozen evidence values are permitted.

## User decision 3 — Phase 3B decision vocabulary

Phase 3B decision vocabulary is restricted to confirmed-cautious or downgraded. The terms upgrade, unchanged, clean, detector-ready, positive class, and any equivalent strengthening language are prohibited. Default rule: if retention cannot be affirmatively defended under the current measurement-model ceiling and unresolved confounders, downgrade conservatively.
